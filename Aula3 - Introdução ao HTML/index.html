O que é o HTML?

HTML é a sigla para HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto.
É uma linguagem de programação que define a estrutura e o conteúdo de páginas web.

Com HTML podemos criar páginas para:

  • Blog
  • Rede social
  • Sites institucionais

Para entender melhor o HTML é preciso:

  1. Entender a estrutura base
  2. Estudar sobre as tags
  3. Praticar os conceitos e pesquisar as dúvidas no Academy
